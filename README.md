# Boids Project
[Video Link](https://drive.google.com/file/d/1N_z6nAQFXvMaQZD08iNhxWwy7yUpmBhb/view?usp=sharing)

A classic implementation of the Boids algorithm in 3D.

There are two added tweaks. A boundary was added that gently nudges the boids back inside it, and a maximum speed was enforced to prevent erratic behavior.
